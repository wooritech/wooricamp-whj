# 변환 (transform)

## 2D 변환 함수

- 단위 없음
``` 
- translate(x,y) 이동 x,y 축
- translateX(x) 이동 x 축
- translateY(y) 이동 y 축
- scale(x,y) 크기 x,y 축
```

- deg 단위
```
- rotate(deg) 회전 각도
- skewX(x) 기울임(x축)
- skewY(y) 기울임(y축)
```
## 3D 변환 함수

- 단위 px
```
- perspective(n) 원근법(거리)
  (rotate보다 우선해서 작성)
```
- deg 단위
```
- rotateX(x) x 축 회전
- rotateY(y) y 축 회전
- rotateZ(z) z 축 회전
```
