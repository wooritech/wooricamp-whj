# CSS 속성

- 너비(width, height)
  - auto: 브라우저가 너비를 걔산 (기본값)
    - 인라인 요소들의 가로 세로 너비를 자동으로 줄이는데 기능
    - 블럭 요소들의 가로 너비를 최대로 늘리고 세로를 줄이는데 기능
- CSS 단위
- 외부 여백(margin)
- 내부 여백(padding)
- 테두리 선(border)과 색상 표현
- 모서리 둥글게(border-radius)
- 크기 계산(box-sizing)
- 넘침 제어(overflow)
- 출력 특성(display)
- 투명도
- 글꼴
- 문자
