# Part 1. ch 3

- Doctype(DTD)
  - !DOCTYPE html 로 시작, 웹표준에 의거하여 실행
  - 문서의 html의 버전을 표시
  - 기본적으로 HTML5를 표준으로 지정
  - HTML1~4는 명시 필요 없음
  - XHTML 버전
    - !DOCTYPE html PUBLIC 으로 시작

- HTML, HEAD, BODY
  - html
    - 문서의 전체 범위를 표시
  - head
    - 비가시적인 "정보"를 나타내는 범위
    - 제목, 설명, 사용할 파일 위치, 스타일(CSS)
  - body
    - 가시적인 "구조"를 나타내는 범위
    - 로고, 헤더, 푸터, 내비게이션, 메뉴, 버튼, 이미지

- CSS, JS 연결하기
- 정보를 의미하는 태그 살펴보기
- 화면에 이미지 태그 살펴보기
- 상대 경로와 절대 경로
- 페이지를 나누고 연결하기
- 모든 파일 공백 크기 설정
- 개발자 도구 사용하기
